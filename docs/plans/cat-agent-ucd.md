## 1. 设计原则

### 1.1 核心理念

**"温暖、友好、高效"** - 让复杂的多Agent协同系统变得亲切易用

| 原则 | 描述 | 应用示例 |
|------|------|----------|
| 温暖友好 | 猫猫主题元素营造轻松氛围 | 可爱的猫爪图标、柔和的配色 |
| 清晰直观 | 信息层级分明，操作路径清晰 | 状态标签明确、按钮位置合理 |
| 一致性 | 全局统一的设计语言 | 统一的组件样式、交互模式 |
| 高效专业 | 不因可爱牺牲专业性 | 数据可视化清晰、操作流程高效 |
| 响应式 | 适配多种设备尺寸 | 桌面端、平板端适配 |

### 1.2 设计风格关键词

`温暖` `可爱` `现代` `简洁` `专业` `灵动`

## 11. 交互动效

### 11.1 过渡动画

| 类型 | 时长 | 缓动函数 | 用途 |
|------|------|----------|------|
| 快速 | 150ms | ease-out | 按钮悬停、图标变化 |
| 常规 | 300ms | ease-in-out | 抽屉展开、模态框 |
| 慢速 | 500ms | ease-in-out | 页面切换、大动画 |

### 11.2 微动画

```css
/* 按钮悬停抬起 */
.btn:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(255, 140, 66, 0.3);
}

/* 卡片悬停放大 */
.card:hover {
  transform: translateY(-4px);
  box-shadow: 0 12px 32px rgba(139, 115, 85, 0.15);
}

/* 图标旋转 */
.icon-spin {
  animation: spin 1s linear infinite;
}

@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
```

### 11.3 猫猫主题动效

```css
/* 猫爪点击效果 */
@keyframes paw-tap {
  0% { transform: scale(1); }
  50% { transform: scale(0.9); }
  100% { transform: scale(1); }
}

.btn:active {
  animation: paw-tap 0.15s ease;
}

/* 猫尾巴摇摆 */
@keyframes tail-wag {
  0%, 100% { transform: rotate(-5deg); }
  50% { transform: rotate(5deg); }
}

/* 成功猫咪动画 */
@keyframes cat-celebrate {
  0% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
  100% { transform: translateY(0); }
}
```

## 13. 无障碍设计

### 13.1 颜色对比度

- 文本与背景对比度 ≥ 4.5:1
- 大文本对比度 ≥ 3:1
- 状态指示不依赖颜色区分（配合图标/文字）

### 13.2 键盘导航

- 所有交互元素可通过Tab键导航
- 焦点状态清晰可见（橙色轮廓）
- 支持快捷键操作
